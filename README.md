# 停车场管理系统 java306

## 项目概述

停车场管理系统是一个基于B/S架构和MVC设计模式的Web应用程序，采用前后端分离的技术架构。系统主要使用Java、SpringBoot、Vue等技术开发，旨在为用户提供便捷的停车服务和管理功能。

## 技术栈

- **后端**: Java、SpringBoot、Mybatis-Plus、Mysql
- **前端**: Vue
- **开发工具**: IDEA、JDK1.8、Maven

## 系统角色

- **用户**: 可以进行登录、注册、个人信息修改、停车类型记录、车位预定与缴费、账户充值等操作。
- **管理员**: 负责管理用户信息、车位信息、费用设置、停泊车辆查询以及车辆进出管理等。

## 功能模块

### 用户功能

1. **用户登录/注册/退出**
   - 用户可以注册新账户，登录系统，或退出登录。
2. **个人信息修改**
   - 用户可以修改自己的个人信息。
3. **停车类型记录**
   - 记录用户的停车类型及近五天的缴费记录统计。
4. **车位预定**
   - 用户可以预定停车位。
5. **车位缴费**
   - 用户可以为预定的车位进行缴费。
6. **账户充值**
   - 用户可以为自己的账户进行充值。

### 管理员功能

1. **用户信息管理**
   - 管理员可以管理用户信息。
2. **车位信息管理**
   - 管理员可以管理停车位信息。
3. **车位费用设置**
   - 管理员可以设置不同车位的费用。
4. **停泊车辆查询**
   - 管理员可以查询当前停泊的车辆信息。
5. **车辆进出管理**
   - 管理员可以管理车辆的进出记录。
